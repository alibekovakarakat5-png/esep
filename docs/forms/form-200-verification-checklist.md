# Форма 200.00 — лист проверки для бухгалтера

> Для Фариды (или любого бухгалтера-партнёра). Цель: подтвердить или
> поправить формулы расчёта. Каждую строку можно поправить в конфиге
> `tax_config` БЕЗ передеплоя приложения.

## Контрольный пример (что выдаёт Esep сейчас)

**Сценарий:** ТОО, 1 работник, оклад **250 000 ₸/мес**, 1 квартал 2026.
МРП = 4 325 ₸, МЗП = 85 000 ₸ (значения из config — проверьте актуальность на 2026).

| Строка | Что | Формула в Esep | Сумма/мес |
|--------|-----|----------------|-----------|
| 200.00.002 | ОПВ | мин(ЗП; 50 МЗП) × 10% | 25 000 |
| 200.00.011 | ВОСМС (взносы работника) | мин(ЗП; 20 МЗП) × 2% | 5 000 |
| 200.00.001 | ИПН | (ЗП − ОПВ − 30 МРП) × 10% | 9 525 |
| 200.00.008 | СО | макс(1 МЗП; мин(ЗП−ОПВ; 7 МЗП)) × 5% | 11 250 |
| 200.00.010 | ООСМС (отчисления работодателя) | мин(ЗП; 40 МЗП) × 3% | 7 500 |
| 200.00.005 | СН (соц. налог) | (ЗП−ОПВ) × 9,5% − СО | 10 125 |
| | **Итого/мес** | | **68 400** |
| | **Итого/квартал** (×3) | | **205 200** |

## Вопросы к бухгалтеру — отметьте ✅ или впишите правку

### 1. ОПВ (обязательные пенсионные взносы)
- [ ] Ставка **10%** — верно?
- [ ] Потолок базы **50 МЗП** — верно?

### 2. ВОСМС (взносы на ОСМС с работника)
- [ ] Ставка **2%** — верно?
- [ ] Потолок базы **20 МЗП** — верно?

### 3. ИПН — **главный вопрос**
- [ ] Вычет **30 МРП** в месяц — верно для 2026? (раньше был 14 МРП / 1 МЗП)
- [ ] Нужно ли вычитать **ВОСМС** из базы ИПН? Сейчас НЕ вычитаем.
      База = ЗП − ОПВ − 30 МРП. **Если надо вычитать ВОСМС → скажите.**
- [ ] Ставка **10%** до порога 8 500 МРП, **15%** выше — верно?

### 4. СО (социальные отчисления)
- [ ] Ставка **5%** — верно?
- [ ] База: ЗП − ОПВ, мин **1 МЗП**, макс **7 МЗП** — верно?

### 5. ООСМС (отчисления работодателя на ОСМС)
- [ ] Ставка **3%** — верно?
- [ ] Потолок базы **40 МЗП** — верно?

### 6. СН (социальный налог) — **второй главный вопрос**
Реформа 2026 по соцналогу — самое неоднозначное место.
- [ ] **Для ТОО:** формула (ЗП − ОПВ) × **9,5%** − СО — верно?
      Или ставка другая? Или без вычета СО? **Впишите правильную:** ___________
- [ ] **Для ИП на ОУР:** фикс **2 МРП за себя + 1 МРП за работника**/мес — верно?

### 7. Общее
- [ ] МРП 2026 = **4 325 ₸** — верно? Если нет: ___________
- [ ] МЗП 2026 = **85 000 ₸** — верно? Если нет: ___________
- [ ] Порядок строк в форме (001=ИПН, 002=ОПВ, 005=СН, 008=СО,
      010=ООСМС, 011=ВОСМС) — совпадает с вашей практикой?

## Как поправить (для разработчика)

Все ставки в `lib/core/constants/kz_tax_constants.dart` + перекрываются
через `tax_config` в БД (правится из админки без передеплоя):

| Параметр | config key | default |
|----------|-----------|---------|
| МРП | `mrp` | 4325 |
| МЗП | `mzp` | 85000 |
| ОПВ работник | `ee_opv_rate` | 0.10 |
| ВОСМС работник | `ee_vosms_rate` | 0.02 |
| ИПН вычет (МРП) | `ipn_deduction_mrp` | 30 |
| ИПН ставка | `general_ipn_rate` | 0.10 |
| СО работодатель | `emp_so_rate` | 0.05 |
| ООСМС работодатель | `emp_vosms_rate` | 0.03 |
| **СН ставка** | `ee_social_tax_rate` | 0.095 |
| СН ИП за себя (МРП) | `ip_sn_mrp_self` | 2.0 |
| СН ИП за работника (МРП) | `ip_sn_mrp_per_employee` | 1.0 |
