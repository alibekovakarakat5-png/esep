# Форма 200.00 — SDK (официальная схема КГД)

**Источник:** официальный пакет КГД РК
**Версия:** 33, ревизия 142
**Период действия:** 2023-01-01 → 2026-12-31

## Описание формы

> Декларация по индивидуальному подоходному налогу и социальному налогу

Сдают:
- ИП на ОУР (общеустановленный режим)
- ТОО (как налоговые агенты)
- Юридические лица с работниками

**Периодичность:** квартальная
**Срок сдачи:** до 25 числа второго месяца, следующего за кварталом

## Файлы пакета

| Файл | Содержимое |
|------|-------------|
| `form_200_00_v33_r142.xml` | Основная схема: 5+ страниц, ~1000 полей с координатами для PDF |
| `form_200_00_v33_r142_chr.xml` | Правила расчёта: формулы, КБК, сроки уплаты, типы документов |
| `form_200_00_v33_r142_loc.xml` | Локализация полей (RU/KZ названия по labelCode/hintCode) |
| `form_200_00_v33_r142_packInfo.xml` | Метаданные пакета (build, дата) |

## Структура формы

Форма 200.00 включает:

- **200.00.01** — титульный лист (реквизиты налогоплательщика, период)
- **200.00.02** — основная расчётная страница
- **200.00.03** — социальные платежи (ОПВ / СО / ВОСМС)
- **200.00.04** — приложение (ИПН с дивидендов и др.)

## Типы декларации (kind codes)

| Код | Значение |
|-----|----------|
| `10000000000000000000` | Первоначальная (inisValue=1) |
| `01000000000000000000` | Очередная (inisValue=2) |
| `00100000000000000000` | Ликвидационная (inisValue=4) |
| `10100000000000000000` | Первоначальная + ликвидационная (inisValue=4) |
| `00011000000000000000` | Дополнительная по уведомлению (inisValue=6) |
| `00010000000000000000` | Дополнительная (inisValue=3) |
