# Форма 300.00 — SDK (официальная схема КГД)

**Источник:** официальный пакет КГД РК
**Версия:** 29, ревизия 170

## Описание формы

> Декларация по налогу на добавленную стоимость (НДС)

Сдают:
- Плательщики НДС (превысили лимит регистрации или зарегистрировались добровольно)
- ИП и ТОО на ОУР с оборотом > порога

**Периодичность:** квартальная
**Срок сдачи:** до 15 числа второго месяца, следующего за кварталом

## Изменения с 2026 (важно!)

С 1 января 2026:
- **Ставка НДС: 16%** (вместо 12%)
- Изменения в порядке возмещения
- Новые требования к ЭСФ
- Усиленный контроль через Webkassa
