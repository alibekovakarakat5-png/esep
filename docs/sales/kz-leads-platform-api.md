# Esep Platform API — целевые лиды по Казахстану (B2B)

**Дата составления:** 2026-05-25.
**Кто составил:** ресёрч-агент по публичным источникам (официальные сайты, 2GIS,
statsnet.co, kompra.kz, kursiv.media, business.gov.kz, новостные порталы).

## Что это за список и как его использовать

Это **холодные high-fit лиды** — компании, чей бизнес-профиль (платформа с
большим числом исполнителей-физлиц) хорошо ложится на наш Platform API:
закрываем обязательства налогового агента по НК 2026 (Закон 214-VIII +
Соцкодекс ст.102, платформенная занятость) — один HTTP-вызов вместо договоров
с КГД и интеграции с ОФД.

**Важно:** «тёплыми» эти лиды НЕ являются. Прошлых касаний с ними у Esep нет.
«Тёплость» появится после первого касания основателя (LinkedIn, звонок, письмо).
Контактные данные брались с **их собственных сайтов и 2GIS** — там, где не нашлось
публичной почты/телефона, поле оставлено пустым (не выдумываем).

**Что специально НЕ включаем:**

- Yandex Eda / Yandex Taxi / Yandex Доставка — у них своя налоговая команда
- inDrive, Maxim, Uber — то же
- Glovo, Wolt, Chocofood — есть собственные compliance-команды и юристы
- Kaspi, Wildberries, Ozon, Halyk Market — enterprise-гиганты, не наш сегмент
- Q Work, Top.kz уровня (см. ниже Top.kz — он у нас включён, но с оговоркой:
  большой, может быть mid-enterprise)

## Как найти ЛПР (decision-maker)

Большинство компаний в этом списке — частные ТОО среднего размера. ЛПР обычно
**основатель / СЕО / Операционный директор / Финдир**.

### Каналы поиска

1. **LinkedIn KZ** — самый эффективный. Поиск по компании + роль:
   - `CEO at <Company>` / `Founder at <Company>`
   - `Operations Director at <Company>` / `Операционный директор`
   - `CFO at <Company>` / `Финансовый директор`
   - `Head of Compliance at <Company>` / `Юрист`
   - **Совет:** многие казахстанские основатели сидят в LinkedIn под англ. именем —
     попробуй оба написания (Айбек/Aibek).
2. **2GIS карточка компании** — иногда отображает прямые телефоны менеджеров и
   ФИО руководителя.
3. **statsnet.co** + **kompra.kz** + **adata.kz** — поиск по БИН/названию,
   возвращает ФИО директора из ЕГРЮЛ, юр.адрес, БИН, иногда учредителей.
   Примеры использования:
   - `https://statsnet.co/companies/kz/<id>`
   - `https://pk.adata.kz/counterparty/main/company/<БИН>/basic-info`
4. **goszakup.gov.kz** — если компания участвует в гос.закупках, виден директор
   и контакты для закупок.
5. **business.gov.kz** — реестр предпринимателей РК.
6. **eGov KZ + Kompra** для проверки актуальности юр.лица и учредителей.
7. **Хабр Карьера** (career.habr.com) — у IT-компаний (Kolesa Group, Naimi.kz)
   часто есть профиль с описанием команды и HR-контактом.

### Подход к ЛПР

- **Сначала** ищем основателя/СЕО в LinkedIn → пишем коротко: «привет, увидел вашу
  компанию, у нас Platform API для НК 2026 по выплатам самозанятым — могу
  показать demo за 15 минут».
- **Если не отвечает** через 3 дня → пишем COO/CFO.
- **Параллельно** — звонок на общий номер, просим «соединить с тем, кто отвечает
  за compliance/налоги по выплатам курьерам».
- **Telegram** — после LinkedIn-ответа просим перейти в TG (там быстрее
  обмениваться).

## Гэпы в исследовании (честно)

1. **Email у 50%+ компаний не нашли** — только сайт-форма или телефон.
   Founder'у придётся либо звонить, либо искать через LinkedIn.
2. **БИН не указан для большинства** — для прицельного поиска по statsnet/adata
   надо вручную пробить каждую (5 мин на компанию).
3. **ФИО ЛПР не указаны намеренно** — публичных источников ненадёжно мало;
   фабриковать = рисковать репутацией. LinkedIn даст актуальный ответ.
4. **Список можно расширить** до 40-50 компаний, если выделить ещё 2-3 часа на
   обзвон 2GIS + поиск по статьям kursiv.media за 2024-2026 (рост платформенной
   экономики РК).
5. **Не охвачены ниши**: грузоперевозки B2B (CMR24, Della, FaFa) — формально они
   биржи, не операторы выплат; но если у них есть white-label режим — можно
   копнуть отдельно.
