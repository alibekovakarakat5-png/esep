# Connect-кампания — 165 SMB-лидов (бухкомпании)

Цель: первые платящие клиенты за неделю через рассылку в WhatsApp.

Канал: Connect, workspace `esep`, инстанс `esep-sales-1`.
Аудитория: 165 лидов (147 с телефонами) — бухкомпании Астаны и др.

---

## Темп (антибан — строго)

Если номер уже прогрет:
- Первый день: **20 сообщений**, дальше по таблице прогрева (CLAUDE.md, раздел 8).
- Между сообщениями — **30-90 сек рандом**.
- Текст варьировать на **30-50%** (Connect это умеет — шаблон с переменными).
- Имя получателя в начале — обязательно.

Сегментация по статусу лида:
- `new` → Сообщение 1 (первое касание)
- `contacted` без ответа 3+ дня → Сообщение 2 (follow-up)
- не ответили после двух → Сообщение 3 (breakup), потом не трогаем.

---

## Сообщение 1 — первое касание

Коротко, по-человечески, про их мир. НЕ продающая простыня.

```
{Имя}, здравствуйте! Вы ведёте бухгалтерию для ИП —
а с 2026 у клиентов всё поменялось: упрощёнка 4%, НДС 16%,
прогрессивный ИПН, патент отменили.

Мы сделали Esep — приложение, которое считает налоги
по новому НК 2026 и формирует ЭСФ. Бухгалтер ведёт
в нём сразу несколько клиентов.

Скинуть короткое демо — посмотреть, подойдёт ли вам?
```

Вариативные куски для Connect (чтобы не было 165 одинаковых):
- «с 2026 у клиентов всё поменялось» / «по новому кодексу всё пересчитывается»
- «считает налоги по новому НК 2026» / «закрывает расчёты по НК 2026»
- «Скинуть короткое демо?» / «Показать, как это выглядит?»

## Сообщение 2 — follow-up (нет ответа 3+ дня)

Новый угол, не «просто напоминаю».

```
{Имя}, добавлю конкретику: в Esep бухгалтер держит всех
своих ИП в одном кабинете — расчёт 910, ОПВ/СО/ВОСМС,
ЭСФ в официальном формате. Без пересчётов вручную
под новый кодекс.

Сезон сдачи на носу. Скинуть демо на 2 минуты?
```

## Сообщение 3 — breakup (последнее касание)

```
{Имя}, не хочу быть навязчивой — закрываю тему.
Если в этом сезоне будет тяжело с пересчётами по НК 2026 —
напишите, демо вышлю в любой момент. Удачной отчётности!
```

---

## Что отвечать на интерес

**«Скиньте / покажите»** → демо-доступ + короткое видео/скрин. Договориться на
10-минутный созвон или дать потыкать самим.

**«Сколько стоит?»** → назвать тариф для бухгалтеров (`accountant_pro`). Если
цена ещё не зафиксирована — «давайте покажу продукт, тариф подберём под объём
ваших клиентов».

**«Я уже работаю в 1С / на бумаге»** → не спорить. «Esep не вместо, а быстрее
именно под новый НК 2026 и ЭСФ. Сравните на одном клиенте.»

**Ответил, но вяло** → перевести статус в `replied`, добавить в тёплый список,
не дожимать рассылкой — личное касание.

## Метрика недели

- Отправлено / доставлено / ответили / заинтересовались / демо показано.
- Цель-минимум: довести 3-5 заинтересованных до демо, 1-2 — до оплаты.
- Связка Connect: статусы лидов вести честно — это база для follow-up.

> Примечание: эти 165 — бухгалтеры (вторичный сегмент). Основная ЦА —
> сами малые предприниматели — это отдельный источник лидов, более долгий.
> Но для первых денег на этой неделе бухкомпании — самый быстрый канал.
