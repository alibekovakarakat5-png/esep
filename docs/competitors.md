# Конкуренты Esep в Казахстане

> Дата: 2026-03-17

## Прямые конкуренты

### 1. e-Salyq Business (государственное)
- **Цена:** Бесплатно
- **Что делает:** Регистрация ИП, оплата налогов, реестр доходов, ф.910
- **Слабость:** Массовые баги, тысячи жалоб, ошибки регистрации, устаревшие ЭЦП, ужасный UX
- **Стратегия Esep:** "То же самое, но работает + больше функций"

### 2. Kaspi Pay / Kaspi Business
- **Цена:** Бесплатно (экосистема Kaspi)
- **Что делает:** Регистрация ИП, оплата налогов, фискальные чеки
- **Слабость:** Не полноценная бухгалтерия — нет документов, ф.910, расчёта зарплаты
- **Стратегия Esep:** "То, чего в Kaspi нет" — полный учёт, отчётность, документы

### 3. BCC Бух (Банк ЦентрКредит)
- **Цена:** Бесплатно (для клиентов BCC)
- **Что делает:** Онлайн-бухгалтерия, оплата налогов, отчётность
- **Слабость:** Привязан к одному банку, ограниченный функционал
- **Стратегия Esep:** Работает с любым банком

### 4. MyBuh.kz
- **Цена:** Платно (тарифы на mybuh.kz/tarif/)
- **Что делает:** Полная бухгалтерия, ЭСФ, зарплата, аутсорсинг
- **Аудитория:** 25,000+ ИП, 300+ компаний/день, Instagram 190K
- **Слабость:** Сложный для обычного ИП, ориентирован на бухгалтеров
- **Стратегия Esep:** Проще, мобильный, для предпринимателя без бухгалтерского образования

### 5. Мой Учет (moyuchet.kz)
- **Цена:** Лайт 2,500₸/мес, Про 4,582₸/мес (при оплате за год)
- **Что делает:** Первичные документы, ф.910, ЭСФ, платёжные поручения
- **Слабость:** Только веб, нет нативного мобильного приложения
- **Стратегия Esep:** Нативное мобильное приложение

### 6. Payda.me
- **Цена:** От 45,000₸ за пакет
- **Что делает:** "Бухгалтерия в смартфоне" — но по факту аутсорсинг через приложение
- **Аудитория:** 5,000+ пользователей
- **Слабость:** Очень дорого, не самостоятельный инструмент
- **Стратегия Esep:** В 20 раз дешевле, даёт автономность

### 7. Asistent.kz
- **Цена:** Платная подписка
- **Что делает:** Облачный бухгалтер для ИП и ТОО, синхронизация ЭСФ, дашборды
- **Слабость:** Менее известен, нет мобильного приложения
- **Стратегия Esep:** Мобильный + известность через Instagram/Telegram

### 8. 1С:Бухгалтерия (облако)
- **Цена:** От 3,500₸/мес (1cfresh.kz, inbuh.kz)
- **Что делает:** Полная бухгалтерская система
- **Слабость:** Сложный интерфейс, нужно обучение, не для ИП-самоучки
- **Стратегия Esep:** "1С — для бухгалтеров, Esep — для предпринимателей"

## Ключевой вывод

**Свободная ниша Esep:** На рынке нет ни одного качественного нативного мобильного приложения для самостоятельного ведения бухгалтерии ИП/ТОО по адекватной цене.

- Бесплатные решения (e-Salyq, Kaspi) — неполные или с багами
- Веб-сервисы (Мой Учет, MyBuh, 1С) — нет мобильного
- Мобильные (Payda) — дорогой аутсорсинг

Esep = нативный мобильный + самостоятельный + дешёвый + простой UX
