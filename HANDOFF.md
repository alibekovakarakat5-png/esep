## 1. КРИТИЧНЫЕ ограничения окружения (Device Guard) — читать перед любой сборкой

Эта машина (Windows, Device Guard/WDAC) **блокирует неподписанные .exe**: `dart`/`flutter`,
`ffmpeg`, Remotion compositor + его chromium, `7z`. Следствия:

- **Flutter локально НЕ компилируется/тестируется.** Деплой app: `git push` в `esep` main →
  GitHub Actions (`build-release.yml`) собирает Flutter web → GitHub Pages (app.esepkz.com),
  ~5–10 мин. ⚠️ `flutter_service_worker.js` — это 784-байтный стаб, НЕ меняется между
  сборками → НЕ индикатор деплоя. Проверять выкатку по строкам в `main.dart.js` (кириллица
  там в `\uXXXX`-эскейпах).
- **Видео локально НЕ рендерятся** (chromium/ffmpeg блокируются). Рендер только в облаке:
  GitHub Actions в репо `studyhub` (`.github/workflows/render-reels.yml`) рендерит Remotion →
  MP4 в **GitHub Release `esep-videos`** → скачиваются в `C:\Users\USER\Downloads\esep-videos\`.
  Remotion **Studio (preview)** локально работает: `cd marketing-video && npx remotion studio
  --port=3003 --no-open` (preview не требует chromium).
- Деплой/пуш — **только по явному «да» Каракат.**
